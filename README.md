# Portfólio — Leonardo Dev

Descrição
- Portfólio pessoal de **Leonardo Soares Tavares**, Frontend Developer focado em React, Next.js e tecnologias modernas de frontend. Este projeto é uma página estática com apresentação de skills, projetos e formas de contato.

Arquivos principais
- [index.html](index.html)
- [style.css](style.css)
- [script.js](script.js)

Funcionalidades principais
- Tema claro/escuro com toggle (desktop e mobile).
- Seleção de idioma com traduções em Português, Inglês e Japonês.
- Menu mobile com overlay e animações.
- Fundo animado com snippets de código flutuantes.

Tecnologias
- HTML5, CSS3 (variáveis CSS), JavaScript (Vanilla).

Como rodar localmente
- Opção 1 — abrir localmente: abra `index.html` diretamente no navegador.
- Opção 2 — servidor simples (recomendado para recursos que dependam de fetch):

  - Com Python 3:
  `python -m http.server 8000`

  - Com Node (serve):
  `npx serve .`

Personalização rápida
- Texto e conteúdo: edite [index.html](index.html).
- Estilos visuais: edite [style.css](style.css).
- Comportamento e traduções: edite [script.js](script.js) no bloco `translations`.
